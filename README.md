# Chat  

Realtime chat webb app to learn `Node.js`, `Angular 4`, `Socket.io`, `Webpack`, `SASS`.

## Features: 

- Realtime chat
- Search and add friends to your friend list
- Image uploads

## Requirements

- Node >= 7.0.0
- MongoDB
- Yarn

## Instalation

- clone this project
- yarn install
- npm start 

[DEMO](https://chatwebb.herokuapp.com)
